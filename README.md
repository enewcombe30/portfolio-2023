## Getting Started

### Prerequisites

- **Node.js** (version 16 or later recommended)
- **npm** (Node package manager)

### 

### Installation

Clone down this repository. You will need `node` and `npm` installed globally on your machine.  

Installation:

`npm install`  

To Start Server:

`npm start`  

To Visit App:

`localhost:3000/Local-Weather`
